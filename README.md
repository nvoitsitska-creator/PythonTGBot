# Telegram Bot
## Introduction
This project is a Telegram bot designed to provide entertainment, learning, and utility features in one place.
It allows users to interact with the bot to take quizzes, get random facts, chat with a simulated star, 
and translate text through Telegram.

## Installation
To set up this project locally, you'll need Python, pip, and a Telegram bot token. 
Follow these steps:

### Clone the repository:
git clone https://github.com/nvoitsitska-creator/PythonTGBot
cd python_telegram_bot
### Install the required dependencies:
pip install -r requirements.txt
## Configuration
Create a .env file in the root directory and add your Telegram bot token:

**BOT_TOKEN**=your_telegram_bot_token_here

## Usage

To run the Telegram bot, execute:

python bot.py

Interact with your bot in Telegram to start exploring features, help with CV, answering quizzes,
translating text, and enjoying fun facts.

## Features
### Random fact: 
Show random interesting facts to the user.
### ChatGPT interface: 
Provide a convenient interface for communicating with ChatGPT.
### Dialog with a star: 
The ability to start a dialogue with your favorite celebrity in an imaginary conversation.
### Quiz:
Interactive quiz with questions, evaluation and scoring.
### Translater:
Fast text translation between different languages.
### CV:
Help you create a CV.
